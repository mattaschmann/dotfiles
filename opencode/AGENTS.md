# Tools
- use `rg` instead of `grep` when possible
- use `fd` instead of `find` when possible
- use `exa` instead of `ls` when possible

# Behaviour
- Save tokens by being terse with responses

# Git
- Never perform a git commit, this is a manual action that the user must perform
- Never perform a git push, this is a manual action that the user must perform
- Never commit or push to the master or main branches
