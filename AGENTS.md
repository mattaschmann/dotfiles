# Dotfiles

Personal dotfiles managed by DotBot. Cross-platform: macOS and WSL2/Linux.

## Install

- `./install` runs DotBot via `install.conf.yaml` (symlinks + shell steps).
- Add new symlinks to `install.conf.yaml` — never symlink by hand.
- `os_specific_links.sh` handles Darwin vs WSL2 links (alacritty, urlview).

## Packages

- `packages.toml` is the single source of truth (apt/brew/cargo/uv/npm/vscode/opencode).
- `Brewfile` is generated by `pkgctl brewfile` and gitignored — never edit or commit it.
- `pkgctl` is the CLI tool for all package operations (update, drift, diff, list, brewfile).
- Run via: `uv run pkgctl <command>` or `./bootstrap.sh <command>`.

### pkgctl commands

| Command | Purpose |
|---------|---------|
| `pkgctl update [MANAGER...]` | Upgrade + ensure declared packages installed |
| `pkgctl drift [MANAGER...]` | Report untracked/missing packages (exit 2 if drift) |
| `pkgctl diff [MANAGER...]` | Show pending upstream changes (opencode, dotbot) |
| `pkgctl list [MANAGER]` | Print what's declared in packages.toml |
| `pkgctl brewfile` | Generate Brewfile from packages.toml |

Flags: `--json` (structured output), `--dry-run`, `--packages-file PATH`.

Managers: apt, brew, cargo, uv, npm, opencode, vscode, dotbot.

## Layout

- Config dirs: `nvim/`, `zsh/`, `tmux/`, `alacritty/`, `starship/`, `tig/`, `rg/`.
- `dotbot/` — submodule, don't edit.
- `src/pkgctl/` — Python package (3.13 + click), the package management CLI.
- `scripts/` — legacy helpers (deprecated, use pkgctl instead).
- `opencode/` — opencode global config, symlinked to `~/.config/opencode/`.
- `.tasks/` — task files.
- `bootstrap.sh` — ensures uv is installed, then proxies to pkgctl.
- `pyproject.toml` — uv project definition for pkgctl.

## Bootstrap (new machine)

1. Clone this repo.
2. Run `./bootstrap.sh update` — installs uv if missing, then runs all package managers.
3. Run `./install` — sets up symlinks and triggers the update.

## Conventions

- Lua formatted with stylua (`.stylua.toml`): 2-space indent, single quotes.
- Don't edit symlinked targets in `$HOME` — edit the source here.
- Python in `pkgctl/` uses Python 3.13, formatted with ruff.
